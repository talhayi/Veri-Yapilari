Ornek2

Ornek3

Ornek5

Ornek6

Ornek7

Ornek8

Ornek9

Ornek10

Ornek11

Ornek12

Ornek13

Ornek14

Ornek15

Ornek16

kesmenin başladığı ve bittiği yer ile kopyalanacak yerin adreslerinin verildğini varsayarak
